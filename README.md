# Quotes App

A modern Android application built with Kotlin, Jetpack Compose, and Firebase Realtime Database that helps users store and manage their favorite quotes from books.

## Features

- Add new quotes
- View saved quotes
- Edit existing quotes
- Delete quotes
- Store book name, author, and page number
- Real-time synchronization using Firebase Realtime Database
- Modern UI built with Jetpack Compose
- Material 3 Design

## Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
- Firebase Realtime Database
- Navigation Compose

## Firebase Setup

1. Create a Firebase project.
2. Enable Realtime Database.
3. Download `google-services.json`.
4. Place the file inside the `app` folder.
5. Configure Firebase dependencies.
